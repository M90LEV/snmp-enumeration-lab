# SNMP Enumeration Lab Simulator

An interactive browser-based lab simulator for practising SNMP 
enumeration techniques, built for CREST CPSA exam preparation.

## About

This lab was built with AI assistance (Claude) as a structured 
revision tool to support my CPSA studies. The SNMP content, attack 
techniques, and defensive controls reflect my own study notes and 
understanding of network enumeration.

## What the Lab Covers

- SNMP version identification (v1, v2c, v3)
- Community string bruteforce with onesixtyone and braa
- Full MIB enumeration with snmpwalk and snmpbulkwalk
- Routing table, ARP cache, and process enumeration
- SNMP SET attacks and TFTP config extraction
- Metasploit SNMP auxiliary modules
- SNMPv3 hardened configuration

## Tools Covered

onesixtyone · snmpwalk · snmpbulkwalk · snmpget · snmpset · 
snmp-check · snmp-enum · braa · nmap · tftp · msfconsole

## Usage

Download `snmp-lab-v2.html` and open in Chrome. No setup required.
